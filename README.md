### 📦 Quick Start
To symlink configurations and install dependencies, run the included bootstrap script:

```bash
git clone [https://github.com/leejunwoo/dotfiles.git](https://github.com/leejunwoo/dotfiles.git) ~/dotfiles
cd ~/dotfiles
chmod +x install.sh
